How the Hybrid System Works:

1. Preview Images (Small): Stored in browser - these are your one-pager images for display
2. PDF Files (Large): Hosted on GitHub - these are your pitch decks and one-pagers for download

Step 2: Upload to GitHub

  1. Go to your GitHub repository (yourname.github.io)
  2. Create a folder called "files"
  3. Upload ALL your PDF files and background image to the "files" folder
  4. Upload index.html to the root
  5. Upload portfolio-data.json to the root

Step 3: Add Projects

  1. Go to "Projects" tab
  2. Upload preview image (PNG/JPG, under 500KB)
  3. Enter pitch deck filename: files/fabrinet-deck.pdf
  4. Enter one-pager filename: files/fabrinet-onepager.pdf
  5. Click "Add to Portfolio"

Step 4: Export & Upload

  1. After adding all projects, click "Export"
  2. Upload the new portfolio-data.json to GitHub
  3. Your live site will load everything!
